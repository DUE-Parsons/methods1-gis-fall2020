Exercises

These exercises supplement Methods 1: GIS.

Please download the exercise data [zip] to follow along. You'll use this data for both days—you only need to download it once.

1: Load Data in QGIS

Practice opening data in QGIS and getting around the interface.

  1. From today's data, open ne_110m_admin_1_states_provinces_lakes by dragging and dropping it into QGIS.
  2. Use the hand tool (Pan Map) to move around, the zoom tools to zoom in and out.
  3. Open the attribute table by right-clicking on the layer in the Layers Panel and clicking Open Attribute Table.
  4. Do the same with nybb.geojson.

2: Styling with QGIS

  1. Open all the data in style-exercise in QGIS. This data is a small excerpt of the data around Union Square in Manhattan.
  2. Experiment with styling each of the layers. Add labels as necessary.
  3. Create a PDF of your map using a Print Layout.

3: Opening CSVs

  1. Go to Layer > Data Source Manager.
  2. Select the Delimited Text tab.
  3. Browse for the CSV file NYPD Motor Vehicle Collisions.
  4. Select CSV for the file format, ensure Longitude and Latitude are selected for the X and Y fields, and EPSG:4326 is selected for the CRS.
  5. Click Add to add the data to your map.

4: Project Projections

  1. Open the Council District Boundaries and ne_110m_admin_1_states_provinces_lakes files in QGIS.
  2. Go to Project > Project Properties.
  3. Select the CRS tab if it's not already selected.
  4. In the Filter text field, type 4326 and select EPSG:4326.
  5. Press OK.
  6. Try it again with 2263 and another projection or two.

5: Categorized maps

  1. Open the streets shapefile in QGIS. You could continue with your style from the excercise 2 if you like.
  2. Style using a Categorized style on the column RW_TYPE. Customize each category's style and label text using the metadata for the file.
  3. Open the NYPD Motor Vehicle Collisions CSV in QGIS if it's not already open.
  4. Style using a Categorized style on the column CONTRIBUTING FACTOR VEHICLE 1.
  5. Try categorizing on some other fields.

6: Selecting data

  1. Open the NYPD Motor Vehicle Collisions CSV in QGIS if it's not already open.
  2. Use Select by expression to select only features where the NUMBER OF PERSONS INJURED field is over 0.
  3. Clear the selection.
  4. Use Select by expression to select only features where the BOROUGH field is equal to BRONX.
  5. Export the selected data as its own shapefile and open it.
  6. Experiment with using Select by expression with another field and/or dataset.

7: Quantitative marker styles

  1. Open the NYPD Motor Vehicle Collisions CSV in QGIS if it's not already open.
  2. Style the points using a Graduated style using the NUMBER OF PERSONS INJURED column.
  3. Experiment with styling by colors and size, try other classification modes.
