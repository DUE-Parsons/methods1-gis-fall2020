Exercises

These exercises supplement Methods 1: GIS.

Please download the exercise data [zip] to follow along. You'll use this data for both days—you only need to download it once.

5: Categorized maps

  1. Open the streets shapefile in QGIS. You could continue with your style from the excercise 2 if you like.
  2. Style using a Categorized style on the column RW_TYPE. Customize each category's style and label text using the metadata for the file.
  3. Open the NYPD Motor Vehicle Collisions CSV in QGIS if it's not already open.
  4. Style using a Categorized style on the column CONTRIBUTING FACTOR VEHICLE 1.
  5. Try categorizing on some other fields.
