8: Attribute joins

In this exercise we are mapping COVID-19 antibody test results. The results are reported by ZCTA, which is an approximation of a zip code. The results are given to us as a CSV with no geographies, so we need to combine the results with a shapefile of ZCTAs.

  1. Open the Modified Zip Code Tabulation Areas (MODZCTA) shapefile in QGIS. This file is from NYC Open Data.
  2. Open the antibody-by-modzcta.csv file in QGIS as a Delimited Text Layer with no geography. This file is from NYC Health on GitHub.
  3. Right click on the spatial layer, and open the layer's Properties.
  4. Select the Joins tab.
  5. Click the + button to begin adding the join.
  6. Fill in the dialog:
    • Join layer: the antibody-by-modzcta.csv file
    • Join field: modzcta_first
    • Target field: modzcta
  7. Click OK.
  8. Style the spatial layer using a graduated style on the percent positive.
